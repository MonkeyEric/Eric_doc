        ```shell
        query_list = [
            User.id > 0,
            User.id <= 3,
        ]
        Session.query(User).filter(*query_list)
        ```
        filter_by 只适合key，value的形式来查询
        ```shell
        re_dict = {}
        if son:
            re_dict['count_type_s'] = son
        Session.query(User).filter_by(**re_dict)
        ````
